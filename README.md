DATA-TALK 旨在通过数据讲述世界


DATA-TALK-TS 是 DATA-TALK 的 TypeScript 版本。
+ 使用 vue3 作为前端框架
+ 使用 vite 作为脚手架
+ 通过 axios 请求后端数据

## Installation
```
git clone https://github.com/HLGhpz/Data-Talk-TS
pnpm install
pnpm dev
```
