# DATA-TALK-TS ![](https://img.shields.io/badge/license-MIT-blue) ![](https://img.shields.io/badge/DATA-TALK-brightgreen)

![](@img/data-talk.png)

DATA-TALK旨在通过数据讲述世界
+ 使用vue3作为前端框架
+ 使用vite作为脚手架
+ 通过axios请求后端数据

## Installation
```
git clone https://github.com/HLGhpz/Data-Talk-TS
pnpm install
pnpm dev
```
